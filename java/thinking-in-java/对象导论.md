Java基本特性：

1. 万物皆为对象
2. 程序是对象的集合，它们通过发送消息来告知彼此所要做的
3. 每个对象都拥有其类型
4. 每个特定类型的所有对象都可以接收同样的消息

# 每个对象都有一个接口

接口确定了对某一特定对象所能发出的请求。

# 每个对象都提供服务

把对象看做一个服务提供者
